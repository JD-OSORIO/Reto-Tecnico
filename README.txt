# Reto-Tecnico
Pequeño proyecto que realice de reto técnico propuesto por empresa de desarrollo.
Objetivo: Realizar una tabla que pueda organizar diferentes productos,
seleccionar y sumar el precio de los productos, ver el registro de los productos y eliminar los seleccionados,
imprimir o descargar con pdf. Todo esta totalmente hecho en Angular sin utilizar una base de datos.
