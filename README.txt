## Requisitos

Instalar Angular, Instalar NodeJS

## Instalación

Abrir la consola de comandos (cmd): cd (ruta-del-proyecto), cd (RetoTecnico), npm i

Para iniciar el cliente en la misma consola: cd (RetoTecnico), ng serve --o
